    По вопросам, касающимся взлома и руководства, обращайтесь сюда

    Обязательно к прочтению

    Этот метод требует компьютер под управлением Linux или Mac.

    Установка h-encore выполняется вручную и требует использования командной строки.

    Прежде чем продолжить, убедитесь, что прочитали всю информацию на странице Установка h-encore

    Что понадобится

    • unpack.sh
    • Демо-версия игры Bitter Smile в формате .pkg
    • Свежая версия h-encore
    • Свежая версия psvimgtools (.zip-архив для вашей ОС)
    • Свежая версия pkg2zip
      • Вам нужно будет скомпилировать pkg2zip вручную, следуя инструкциям в readme
    • Свежая версия QCMA
      • Если ранее было установлено официальное приложение Sony Content Manager Assistant, удалите его
      • Пользователи MacOS должны скачать “noffmpeg” версию QCMA

    Инструкция

    Часть I - Подготовительные работы

    1. Установите и запустите на компьютере QCMA, если вы еще этого не сделали
    2. Скопируйте содержимое zip-архива auto h-encore в папку на вашем компьютере
    3. Запустите на консоли приложение “Управление данными”
    4. Выберите “Скопировать контент”
    5. Выберите “Компьютер”
    6. Выберите метод, который вы хотите использовать для подключения к QCMA
      • Если вам предложат войти в аккаунт PlayStation Network, сделайте это
      • Если у вас нет аккаунта PlayStation Network, создайте его
    7. Выберите / зарегистрируйте ваш компьютер, если появится запрос

    Часть II - Установка h-encore

    1. Перейдите по ссылке cma.henkaku.xyz на компьютере
    2. Перейдите в папку PS Vita/APP/ на компьютере с помощью проводника
      • По умолчанию эта папка находится внутри папки Documents на вашем компьютере
      • Если вы не можете её найти, проверьте настройку “Applications / Backups” в приложении QCMA
    3. Скопируйте имя папки внутри этой директории, затем вставьте его в поле “AID” на сайте CMA
      • Вот пример AID: ec8d45ec9c055609
    4. Нажмите “Отправить запрос” для создания соответствующего ключа
      • Оставьте эту страницу открытой; ключ вам вскоре понадобится
      • Вот пример ключа: 488d3ff6cce7a9ef1be6da8338c7d18db18147c3255fa4d50596568555687174
    5. Скопируйте папку h-encore из .zip-архива h-encore в папку на вашем компьютере
    6. Скопируйте содержимое zip-архива psvimgtools в папку h-encore
    7. Скопируйте скомпилированный запускаемый файл pkg2zip в папку h-encore
    8. Переместите .pkg-файл Bitter Smile в папку h-encore
    9. Скопируйте unpack.sh в папку h-encore
    10. Откройте окно терминала
    11. Перейдите в папку h-encore командой cd
    12. Выполните sh unpack.sh
    13. Выполните ./psvimg-create -n app -K YOUR_KEY app PCSG90096/app
      • Замените “YOUR_KEY” на ключ, созданный ранее
    14. Выполните ./psvimg-create -n appmeta -K YOUR_KEY appmeta PCSG90096/appmeta
    15. Выполните ./psvimg-create -n license -K YOUR_KEY license PCSG90096/license
    16. Выполните ./psvimg-create -n savedata -K YOUR_KEY savedata PCSG90096/savedata
    17. Закройте окно терминала
    18. Скопируйте папку h-encore/PCSG90096 в папку PS Vita/APP/YOUR_AID/ на вашем компьютере с помощью проводника

    Часть III - Перенос данных QCMA

    1. Нажмите правой кнопкой мыши на значок QCMA в трее, затем выберите “Refresh database”
    2. В приложении Управление данными на консоли выберите “PC -> Система PS Vita”
    3. Выберите “Приложения”
    4. Выберите “PS Vita”
    5. Выберите “h-encore”
    6. Выберите “Копировать”
    7. Выберите “OK”
      • Эксплойт h-encore будет скопирован на вашу консоль
      • Этот процесс займет некоторое время
    8. Закройте приложение Управление данными
